# Projeto Escola PostgreSQL

Projeto criado para praticar conceitos fundamentais de Banco de Dados utilizando PostgreSQL.

## Conceitos praticados

* CREATE TABLE
* PRIMARY KEY
* FOREIGN KEY
* INSERT
* SELECT
* UPDATE
* DELETE
* WHERE
* LIKE
* ORDER BY
* INNER JOIN
* GROUP BY
* HAVING
* COUNT
* ON DELETE CASCADE
* ON UPDATE CASCADE

## Estrutura do banco

### aluno

* id
* nome
* idade

### curso

* id
* nome
* valor

### aluno_curso

Tabela responsável por relacionar alunos e cursos (N:N).

* PostgreSQL
